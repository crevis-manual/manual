Quick Start

  1. Connect GigEStrobe to the host PC using a qualified Ethernet cable.
  2. Apply power and confirm that the device status indicators are normal.
  3. Set the PC network adapter so it shares the same subnet as the camera.
  4. Launch the control software and search for the connected device.
  5. Adjust strobe timing, trigger options, and save the final configuration.

Setup Notes

Use a stable power source, verify IP settings first, and confirm that exposure and trigger timing are aligned before validating strobe behavior. This usually removes the most common first-run issues.

If the device is not detected, check firewall rules, adapter IP range, and cable integrity before changing application settings.
